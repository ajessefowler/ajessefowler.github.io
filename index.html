About

Captivated by computers since I was four years old, I've grown into a versatile and detail-oriented web developer. While attending Columbus State Community College, I established a solid foundation by familiarizing myself with fundamentals such as object-oriented programming, agile development, networking, and database design. I graduated in August 2018 with a 3.75 GPA.

Leading a team through the design and implementation of a full stack application for CSCC's capstone project allowed me to develop my leadership skills and gain experience delivering effective, efficient business solutions. To explore my interest in user interface design, I created a minimalistic yet fully featured weather reference website. Implementing features such as a location history with favorite locations and an interactive radar map challenged and refined my front end proficiency.

Projects

Speedcats Car Rental

Django, PostgreSQL, JavaScript

Built for Columbus State Community College's capstone project, this application modernizes a fictional rental car business's inventory and reservation systems. A straight-forward, responsive design encourages users to place reservations by ensuring the user experience is as quick and frictionless as possible. Employees' workflows are significantly improved by an automated inventory system and a fully featured administrative site.

Thunderbolt Weather

HTML, CSS, JavaScript, Node.js

A modern, minimalistic take on a fully featured weather reference. An intuitive design centered around an interactive radar map minimizes the time it takes to understand the current weather situation. Viewed locations are automatically saved, and users may select a favorite location to load by default. Fluid animations make the user experience more satisfying without sacrificing speed or ease of use.
